●このライブラリにおけるテストの基本的な考え方
※テストの一塊としての最小単位をブロックと定義
Count   : ブロック数
State   : ブロックそれぞれのテストの状態
Failure : ブロックと対応するテスト失敗の数
Total   : テストの総判定数
Error   : ブロック内の各テストの判定でエラーと判断された数
Warning : ブロック内の各テストの判定で警告と判断された数
Notice  : ブロック内の各テストの判定で通知と判断された数
